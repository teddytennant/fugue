# CLI Reference

Fugue provides a command-line interface for managing configuration, plugins, and the vault.

## Commands

- `fugue init` -- Initialize a new configuration file.
- `fugue run` -- Start the Fugue daemon.
- `fugue vault set <key>` -- Store a credential in the vault.
- `fugue vault get <key>` -- Retrieve a credential from the vault.
- `fugue plugin install <name>` -- Install a plugin.
- `fugue plugin list` -- List installed plugins.

Run `fugue --help` for full usage information.
